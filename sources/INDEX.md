- [Multimodal Attacks](#multimodal-attacks) — 26 papers, 175 exploit entries. Image, audio, video, OCR, typographic, and vision-language prompt injection / jailbreaks against MLLMs.
- [Training Poisoning & Backdoors](#training-poisoning-backdoors) — 20 papers, 97 exploit entries. Training-data poisoning, backdoor triggers, sleeper agents, RAG/index poisoning, model supply-chain.
- [Human Manipulation](#human-manipulation) — 27 papers, 3 exploit entries. LLMs manipulating humans: persuasion, dark patterns, sycophancy toward users, scams / smishing / vishing, deceptive UX.
- [Deception & Alignment](#deception-alignment) — 11 papers, 3 exploit entries. Model scheming toward developers/evaluators: alignment faking, sandbagging, situational awareness, CoT monitorability, reward hacking.

- [Defenses & Benchmarks](#defenses-benchmarks) — 35 papers, 0 exploit entries. Detection methods, safety benchmarks, guardrail systems, red-team frameworks, constitutional AI, evaluations of defenses.
- [Surveys & Systematizations](#surveys) — 32 papers, 0 exploit entries. Surveys, SoKs, taxonomies, threat-model overviews.
