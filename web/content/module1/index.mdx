# Module 1: The AI Agent Threat Landscape

## Learning Objectives

By the end of this module you will be able to:

1. Explain why AI agents create a new threat class — and why most compromises start with ordinary public inputs, not sophisticated hacks.
2. Identify the three structural properties that make AI agents uniquely vulnerable.
3. Describe what the "confused-deputy problem" means in the context of AI agents.
4. Name the 10 enterprise workflow categories that form the AI attack surface and the public surface that feeds each one.
